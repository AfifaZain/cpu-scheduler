Comparison with Android CFS

Simulated Algorithms: These are static. They rely on fixed values (Burst time) known in advance (which is impossible in real OS) or simple queues.

Android CFS (Completely Fair Scheduler): Real-world Android uses CFS. It does not use fixed time slices. Instead, it uses vruntime (virtual runtime). It tracks how long a process has run on the CPU. The process with the lowest vruntime gets the CPU next. This ensures fairness over time, preventing starvation, unlike Priority scheduling which can starve low-priority tasks.
